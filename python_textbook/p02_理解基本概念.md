## 变量

- 变量就是可以变化的量，是计算机语言中能表示值的抽象概念。
- 变量的定义由三部分组成：
  1. 变量名：指向数据所在的内存地址，通过用它访问数据。
  2. 等号（=）：赋值符号，用来将数据所在的内存地址绑定给变量名。
  3. 值：内存地址对应空间存储的数据。
- 变量需要先定义再使用，解释器执行到变量定义的代码时会申请内存空间存放数据，然后将数据所在的内存地址绑定给变量名。
- 标识符：简单来说，标识符就是你在代码中给各种“东西”起的名称
- 变量命名规范（其它标识符也是类似）：
  1. 变量名通常只包括字母、数字和下划线，且不能以数字开头。
  2. 变量名是区分大小写的，如：num 和 Num 是两个不同的变量名。
  3. 由多个单词构成的变量名，可用下划线来分隔，或使用驼峰命名法。
  4. 变量名要尽量简短及具描述性，尽量做到短小精悍，见名知义。
  5. 不能使用关键字作为变量名。
  6. 尽量不要使用内置函数或内置类的名字作为变量名。

```python
a = 789
b = a
print(a)
print(b)

a = 345
print(a)
print(b)

b = a
print(a)
print(b)
```



## 注释

- 在Python中，注释是用来在代码中添加说明的文字，它不会被Python解释器执行。

- 单行注释：以 # 开头来实现

- 多行注释：以成对的三个引号来实现

```python
# 定义变量，a是变量名，=是赋值符号，2是值
a = 2  # 定义变量，a是变量名，=是赋值符号，2是值

"""
定义变量可以让数据的使用更方便，
因为变量定义好之后，就可以通过
变量名代替该数据反复多次的使用
"""

'''
下文就是对变量名a所指代的数据
进行多次的算术运算
'''
print(a + 4)
print(a - 4)
print(a * 4)
print(a / 4)
print(a ** 4)
```



## 错误

- Python中的错误主要分为两大类：**语法错误**和**运行时异常**。
- **语法错误**：发生在代码编写阶段，因不符合Python语法规则导致解释器无法解析代码
- **运行时异常**：发生在代码执行的过程中

```python
a = 5
print(a + 6)
print(a + 7)
print(a *** 2)  # 语法错误
print(a + 8)
```

```python
a = 5
print(a + 6)
print(a + 7)
print(a / 0)  # 运行时异常
print(a + 8)
```



## 关键字

- 在Python中，关键字是一组预定义的标识符，它们具有特定的语法功能和用途，这些关键字在Python的语法中扮演着至关重要的角色。例如：def用于定义函数，class用于定义类，if用于条件语句等。

```python
# 关键字def用于定义函数
def add(a, b):
    print(a + b)


# 关键字class用于定义类
class Student:
    attr = 789


# 关键字if用于条件语句
c = 2
if c > 0:
    print(8 / c)
```



## 缩进

- 行开头处的空格就是缩进，用来表示代码的逻辑层级，一次缩进通常是4个空格（Tab / Shift + Tab）
- 缩进不光是为了代码美观和可读性，更是Python语法的一部分，不正确的缩进可能影响代码的执行逻辑，甚至导致报错。
- 缩进通常出现在复合语句中，例如：定义函数，定义类，条件语句等。

```python
"""
复合语句的代码通常会占用多行，此时就
需要通过缩进来表示这多行代码的逻辑层级
"""
# 定义函数
def add(a, b):
    print(a + b)


# 定义类
class Student:
    attr = 789


# 条件语句
c = 2
if c > 0:
    print(8 / c)
```



## 函数

- 函数让程序更加模块化，可读性更强且易于复用
- 形参：形式参数，函数在定义时声明的参数
- 实参：实际参数，函数在调用时传入的参数
- 函数体：函数的具体内容，它决定了函数的功能
- 函数在定义时，不会执行函数体；当函数被调用时，才会执行函数体
- 函数只需要定义一次，就可以被多次调用
- 函数每次被调用时，都会把实参传递给对应的形参，然后执行函数体

- 定义函数的语法

  def func_name(par1, par2, ..., parN):

  ​       func_body

  

- 调用函数的语法

  func_name(arg1, arg2, ..., argN)

```python
def f(x, s, w):
    print(2 * x + 3 * s - w + 4)


"""
传位置参数: 实参会按照从左往右的顺序依次传递给形参
"""
f(1, 2, 3)
f(2, 1, 3)
f(3, 2, 1)


"""
传关键字参数: 实参会根据指定的名称传递给同名的形参(和位置顺序无关)
"""
f(x=1, s=2, w=3)
f(x=1, w=3, s=2)
f(w=3, s=2, x=1)


"""
注意: 所有的关键字参数必须放在所有位置参数的后面
"""
f(1, 2, w=3)
f(1, s=2, w=3)
f(1, w=3, s=2)


def f(x, s, w):
    print(2 * x + 3 * s - w + 4)
    print(x - s)
    print(s ** w)
    print(x - w - 100)


f(1, 2, 3)
f(2, 1, 3)
f(3, 2, 1)
```

- 函数是否有返回值，取决于关键字return
- 如果函数没有返回值，等价于return None
- return的作用：将它后面的对象返回给函数调用方，函数调用方就变成了该对象，并且将所在的函数结束。

```python
def f(x, s, w):
    return 2 * x + 3 * s - w + 4


res = f(1, 2, 3)
print(res)
print(res ** 2)
print(res / 3)
print(res + 100)

print(f(1, 2, 3))
print(f(1, 2, 3) ** 2)
print(f(1, 2, 3) / 3)
print(f(1, 2, 3) + 100)


def f(x, s, w):
    print(2 * x + 3 * s - w + 4)


res = f(1, 2, 3)  # res = None
print(res)  # print(None)

print(f(1, 2, 3))  # print(None)
```

- **内置函数**：官方在builtins模块顶层定义好的函数，它们可以直接拿来使用，例如: print，input，len等
- **内置类（类对象，类型）**：官方在builtins模块顶层定义好的类，它们可以直接拿来使用，例如: type，int，str等
